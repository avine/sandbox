jQuery.fn.biMenu, jQuery.fn.biNavMenu

Gestion des menus de l'interface utilisateur.

$(selector).biMenu({
	closeDelay: 750, // Set false to disable this feature
	breadcrumbDisabled: false, // "State less" behaviour
	breadcrumbAlwaysVisible: false,
	callback: function (e, details) { },
	tags: ['ul', 'li']
});

Le plugin biMenu est conçu pour être utilisé avec le plugin biWrap.
Vous pouvez "out-of-the-box" afficher des menus adaptés aux régions suivantes de l'interface :

  • .bi-top : Plusieurs sous-niveaux autorisés.
  • .bi-sidebar : Plusieurs sous-niveaux autorisés.
  • .bi-bottom : Un seul sous-niveau autorisé.
  • .bi-head : Pas de sous-niveau autorisé.
Options
Nom Valeurs Description
closeDelay int (ms), false Définie en ms le temps de latence avant la fermeture automatique des éléments ouvers du menu.
breadcrumbDisabled false, true Définie un menu sans état, à savoir qu'il n'y a pas d'éléments actifs.
breadcrumbAlwaysVisible false, true Empêche la fermeture des éléments actifs du menu.
callback function Fonction de rappel unique pour l'instance executée à chaque action sur le menu.
tags [ParentNode, ChildNode] Renseigne sous forme de tableau de chaînes les noms des balises uniques à utiliser pour générer le menu.
Méthodes
Nom Paramètres Retour Description
breadcrumb object Retourne un object indiquant les noeuds du DOM impliqués dans le fil d'ariane.
breadcrumbArray sep array, string Retourne le fil d'ariane sous la forme d'un tableau de chaînes. Si le paramètre sep est renseigné alors la méthode retourne une chaîne.
close jQuery object Ferme les éléments ouverts du menu.
open item jQuery object Ouvre un élément du menu. item peut être un noeud du DOM ou un entier correspondant à l'index de l'élément à ouvrir.
rebuild jQuery object Rafraîchie l'état du menu.
addItem param, callback DOM node Ajoute un élément au menu configuré avec l'objet param et exécute la fonction callback à chaque action sur cet élément. param accepte les propriétés suivantes : { menu: DOM_NODE, content: DOM_NODE, attributes: {} }. Si la propriété menu n'est pas renseigné, le nouvel élément est ajouté à la racine.
removeItem item, menu DOM node item peut être un noeud du DOM ou un entier correspondant à l'index de l'élément à retirer. menu renseigne le menu parent contenant l'élément à retirer.
Evénements
Nom Details Description
menu.biEvent { item: DOM_NODE, previousItem: DOM_NODE } L'événement est déclenché à chaque action sur le menu. 
$(selector).biNavMenu();

Le plugin biNavMenu est conçu pour la région .bi-top uniquement.
Il permet de rendre le menu de cette région rétractable sur les écrans "small" (dont la largeur est inférieure à 980px).
