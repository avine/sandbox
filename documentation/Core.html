Bi.Core

La classe Bi.Core permet de réaliser dans une syntaxe procédurale des chaînes de traitements asynchrones, sans passer par des fonctions de rappel imbriquées (dans l'esprit de l'interface Promise).

Cette classe est l'ossature de l'application Bi et permet d'organiser ses différentes fonctionnalités en modules.

Sur le plan technique, vous devez parfaitement maîtriser le fonctionnement de cette classe ainsi que ses différentes méthodes, pour pouvoir intervenir efficacement sur le code source de l'application. 

var core = new Bi.Core();

Instancie une nouvelle pile d'appels.
