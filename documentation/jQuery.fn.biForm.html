Gestion des champs de formulaire

<form class="bi-form">
    <!-- Label + Input -->
    <label>Text</label><input type="text" />
    <!-- Input + Label -->
    <input type="checkbox" /><label>Text</label>
    <!-- Label > Input -->
    <label>Text<input type="text" /></label>
    <!-- Label > Input (with label text after input) -->
    <label class="bi-form-nested-inverted"><input type="checkbox" />Text</label>
</form>

De manière générale, vous pouvez afficher le label avant ou après le champs.
Vous pouvez également imbriquer le champs à l'intérieur du label. 

var addAction = window.demoFormActions();
addAction('title', function () {
    // Callback...
});

La fonction globale demoFormActions est dédiée à cette documentation.
Elle permet d'ajouter simplement des boutons d'action avec un title et un callback.
Vous devez ajouter la balise <div id="demo-form-actions"></div> où seront insérés les boutons d'action. 

window.demoFormActions()('title1', callback1)('title2', callback2);

Il est possible de chaîner les appels pour ajouter plusieurs boutons d'action.
