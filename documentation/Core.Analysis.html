Bi.Core.Analysis

Gère l'affichage des analyses.

var analysis = new Bi.Core.Analysis();

Crée une instance de Analysis.

A l'attention des développeurs de rapport BI :
Concernant le paramètrage des Analysis, il vous faudra en cas de besoin regarder le code source pour savoir comment utiliser tel ou tel attribut...

Dans le script /dashboard/app/bo/Analysis.js, consultez les fonctions preprocessCommunAttr et preprocessChartAttr, qui traitent les attributs globaux des analyses.

Dans le script /dashboard/app/bo/AnalysisDisplay.js, consultez pour chaque type de rendu l'objet attributes (présent dans la signature de la fonction de rendu), qui permet d'acceder aux paramètres de l'analyse.
