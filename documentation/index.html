Les composants métiers de l'application se trouvent dans le répertoire : /dashboard/app/bo (Business Object).
Les composants graphiques de l'application se trouvent dans le répertoire : /dashboard/app/ui (User Interface).

L'application fonctionne en mode SPA (Single Page Application).
La page principale est la vue dynamique /Views/Dashboards/Index.cshtml, qui se base sur le template /Views/Shared/_Layout.cshtml. Lorsque le site est déployé en mode "release", les scripts sont compressés en Bundle, et cela peut rendre difficile le debogage éventuel. Si besoin, vous pouvez à tout moment consulter la page statique alternative à l'adresse /dashboard/index.html. Notez que dans ce case, certains paramètres du Web.config n'auront pas d'effet (n'étant pris en charge que dans la vue dynamique).

Pour étendre la documentation, consultez l'aide à la documentation.
