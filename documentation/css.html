CSS

Les feuilles de styles de l'application Bi sont écrites en Less.

Les fichiers Less sont organisés dans le répertoire : /dashboard/app/ui/less.

Pour compiler les fichiers Less, nous avons utilisé le logiciel Koala, qui a l'avantage de proposer une option Autoprefix (ajout automatique des préfixes constructeur -webkit-*, -moz*-, -ms-*).

Le répertoire des feuilles de styles est : /dashboard/app/ui/css.
Il contient uniquement 2 fichiers: la feuille de styles compilée bi.css et une feuille de style minimaliste fullpage.css.

N'éditez jamais la feuille de style compilée, mais compilez toujours le fichier Less source: /dashboard/app/ui/less/bi.less.

Si vous intégrez l'application Bi comme composant d'une page HTML plus globale alors utilisez uniquement la feuille de styles compilée.
Si l'application Bi est affichée en "pleine page" alors ajoutez la feuille de style minimaliste fullpage.css.
