jQuery.fn.biWrap - Standalone

La classe .bi-standalone permet d'afficher uniquement le contenu principal de l'interface (c'est-à-dire le contenu de la balise .bi-body).
Dans ce cas, les régions comme .bi-top, .bi-sidebar, ... sont masquées.

Cette fonctionnalité est utile lorsque vous devez intégrer le contenu principal de l'interface dans une page HTML plus globale.
