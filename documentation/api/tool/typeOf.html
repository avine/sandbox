Bi.tool.typeOf

Teste et détermine le type d'une variable Javascript.

Bi.tool.typeOf(data[, type]);

Retourne le type de la variable data sous forme de chaîne en minuscules, par exemple: "string".

Si le paramètre type est renseigné, alors la fonction retourne un booléan.
type peut-être une chaîne ou un tableau de chaînes.
La fonction retourne true si le type de data correspond à l'un des type attendus et false sinon. 

Bi.tool.is.boolean(data);

Bi.tool.is.number(data);

Bi.tool.is.string(data);

Bi.tool.is.array(data);

Bi.tool.is.object(data);

Bi.tool.is.function(data);

Retourne true si data correspond au type attendu et false sinon.
