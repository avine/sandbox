Bi.tool.console

Affiche des messages dans la console du navigateur (window) ou directement dans la page HTML (document).

Bi.tool.console.log([arg1[, arg2[, ...]]]);
Bi.tool.console.warn([arg1[, arg2[, ...]]]);
Bi.tool.console.error([arg1[, arg2[, ...]]]);

Affiche des messages dans la console courante.

Bi.tool.console.type(["window"|"document"]);

Retourne le type de la console courante : "window" ou "document".
La fonction accepte un paramètre et permet de modifier le statut de la console courante.

Si la console courante est "window" alors la fonction Bi.tool.console.log (et similaires) accepte jusqu'à 5 arguments.
Sinon le nombre d'arguments est illimité. 

Bi.tool.consoleWindow.log(); // .log, .warn or .error
Bi.tool.consoleDocument.log(); // .log, .warn or .error

Force l'affichage dans la console spécifiée, quel que soit le statut de la console courante.

Exemple :

Switch console.type  

console.log   console.warn   console.error

Force window console   Force document console
