Bi.Core.Analysis : les attributs de base

Base
TitleRenseigne le titre de l'analyse.
ZoomAffiche un bouton pour zoomer l'analyse.
RefreshIndique si l'analyse peut être automatiquement rafraîchie (par exemple lorsque la taille de la fenêtre a changé).
Choix et jointure de Dataset
DatasetNom du Dataset sur lequel est basé l'analyse.
DatasetJoinLorsqu'une analyse est basée sur plusieurs Dataset, indique la manière de lier ces Datasets.
GroupRender...sert actuellement à afficher des Sparkline...
Manipulation du Dataset
DatasetSortTri les lignes du Dataset
DatasetFormatApplique un formatage basé sur les colonnes du Dataset.
On suppose dans ce cas que les colonnes choisies possèdent une valeur unique.
Prep-rocessing global
ColumnExpressionBeforeRowSplitterAjoute des colonnes au Dataset, basées sur un calcul. Cette étape est réalisée avant le RowSplitter.
RowSplitterTransorme des colonnes spéciales (qui contiennent plusieurs données distinctes) en lignes.
RowSplitterLimitApplique une limite inférieure et supérieure lors du RowSplitter.
ColumnExpressionAjoute des colonnes au Dataset, basées sur un calcul. Cette étape est réalisée après le RowSplitter.
RemoveEmptyRowsRetire du Dataset des lignes considérées comme ne contenant pas de données pertinante.
Prep-rocessing des Charts
PivotGraphsColorsFromDataSourceDétermine la couleur des graphes en se basant sur un calcul dynamique.
AxisXColonne sur laquelle est basé l'axe X du graphe.
AxisXTitleTitre de l'axe X.
AxisYListe des colonnes du Dataset à afficher dans le graphe sur l'axe Y.
AxisYTitleTitre de charque graphe sur l'axe Y.
PivotYRemplace AxisY et indique que les données doivent être pivotées.
PivotIndicatorsRenseigne les indicateurs qui serviront de graphes à afficher sur l'axe Y.
