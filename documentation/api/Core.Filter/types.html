Bi.Core.Filter : types de filtres et paramètrage

Input
typetext (default) | number | rangeIndique le type d'Input, ce qui permet d'utiliser les nouveaux Input HTML5.
Calendar
A noter que la date courante du calendrier se base sur la VALUE (et non la CAPTION) retournée par le Dataset du filtre.
Cette date doit être de la forme yyyyMMdd.
A noter que vous pouvez afficher un calendrier en utilisant le fitre de type Input.
Mais dans ce cas, vous ne pourrez pas fixer la valeur courante du calendrier.
Radio / Checkbox
orientationvertical (default) | horizontalIndique l'orientation des options du filtre.
Select
size3, 4, 5, ...Attribut optionnel. S'il n'est pas renseigné alors le filtre est de type dropDown.
Sinon, la boîte possède une hauteur fixe en nombre de lignes.
uncheckRadio0 (default) | 1Uniquement lorsque le filtre est de type "radio", permet de dé-sélectionner l'option sélectionnée.
ButtonGroup
typeradio (default) | checkboxRenseigne le comportement du filtre.
uncheckRadio0 (default) | 1Uniquement lorsque le filtre est de type "radio", permet de dé-sélectionner l'option sélectionnée.
vertical0 (default) | 1Renseigne l'orientation des options du filtre.
fullWidth1 (default) | 0Uniquement lorsque le filtre est orienté horizontalement, indique si les boutons doivent remplir toute la largeur disponible.
fixedWidth1 (default) | 0Uniquement lorsque le filtre remplie toute la largeur disponible (fullWidth=1), indique si les boutons doivent avoir la même largeur.
TreeView
typeradio (default) | checkboxRenseigne le comportement du filtre.
uncheckRadio0 (default) | 1Uniquement lorsque le filtre est de type "radio", permet de dé-sélectionner l'option sélectionnée.
threeStates0 (default) | 1Uniquement lorsque le filtre est de type "checkbox", permet de sélectionner plusieurs options à la fois.
dropDown0 (default) | 1Indique si le filtre est de type "dropDown" ou non.
closeButton0 (default) | 1Uniquement lorsque dropDown=0, permet de plier/déplier les options.
uncheckButton0 (default) | 1Uniquement lorsque le filtre est de type "checkbox", permet de sélectionner/dé-sélectionner toutes les options.
maxHeight200pxAttribut optionnel. Quel que soit la valeur de dropDown, permet de donner au conteneur une hauteur maximale des options du filtre.
