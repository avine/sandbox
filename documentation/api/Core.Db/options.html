Bi.Core.Db

var db = new Bi.Core.Db([{
    debug: false
}]);

Crée une instance de Db.

Le paramètre debug active le débogage de l'instance et permet d'obtenir des informations comme la durée d'exécution des requêtes.

Option de débogage : Pour émuler une navigateur lent ou une connexion serveur ayant des temps de réponse longs, vous pouvez modifier le code source du script Core.Db.js en fixant une valeur différente de zéro (en millisecondes) à la variable randomDelay.
