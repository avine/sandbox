Bi.Core.Factory

La fabrique instancie les modules nécessaires au fonctionnement de l'application, et met à disposition un mécanisme permettant aux différentes instances de communiquer entre elles (injection de dépendances).

Potentiellement, il est possible dans une même page Web, d'intégrer deux instances de l'application parfaitement isolées.

var factory = new Bi.Core.Factory();

Crée une instance de Factory.
