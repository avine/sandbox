jQuery.fn.biScreen

Détection du "breackpoint" CSS sur la largeur d'écran (responsive design).

$(selector).biScreen({
    callback: function (e, status) { /* do something... */ }
});

Lorsque la largeur est supérieure ou égale à 980px alors l'écran est considéré comme "large", sinon il est "small".

<!-- The class "bi-screen-large" (or "bi-screen-small") is added dynamically -->
<body class="bi-screen-large">
    <script>jQuery('body').biScreen();</script>
</body>

Dans cet exemple, le plugin est appliqué sur la balise body qui reçoit la classe dynamique body.bi-screen-small ou body.bi-screen-large en fonction de la largeur de la fenêtre.

Options
Nom Valeurs Description
callback function Définie la fonction de rappel à invoquer chaque fois que l'écran change de type de largeur.
Méthodes
Nom Paramètres Retour Description
status "small" ou "large" Retourne la valeur courante du type de largeur d'écran.
Evénements
Nom Details Description
screen.biEvent "small" ou "large" L'événement est déclenché à chaque changement de type de largeur.
