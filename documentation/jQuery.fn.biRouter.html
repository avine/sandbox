jQuery.fn.biRouter

Moteur de pages dans une "single page application" (SPA).

$(selector).biRouter({
    skipAnimation: false,
	slide: false,
	background: false,
	static: false,
	nodeName: ''
});
Options
Nom Valeurs Description
skipAnimation false (default), true Désactive l'animation CSS à l'instanciation du plugin.
slide false (default), true Ajoute un effet "slide" au changement de page.
background false (default), true Ajoute un couleur de fond au conteneur global et aux conteneurs des pages.
static false (default), true Utilise la propriété CSS display:none; pour cacher les pages inactives.
Par défaut, les pages sont cachées en jouant sur les propriétés position:absolute; et z-index.
nodeName "div" (default), "span" Renseigne le type de balise contenant chaque page.
Pour obtenir un comportement "inline", utiliser nodeName="span" et static=true.
Méthodes
Nom Paramètres Retour Description
add path, html, makeActive DOM node Ajoute une page au router. path représente l'identifiant (ou chemin) de la page. html représente le contenu de la page sous forme de chaîne ou de noeud du DOM. La méthode retourne le noeud du DOM contenant la nouvelle page.
remove path DOM node (ou null) Retire la page du router correspondante au chemin path. La méthode retourne le noeud du DOM contenant la page retirée.
goTo path DOM node (ou null) Rend active la page du router correspondante au chemin path. La méthode retourne le noeud du DOM contenant la page atteinte.
getPath   string Récupère le chemin de la page active.
isPath path boolean Retourne true si le chemin renseigné correspond à une page du router.
getContainer path DOM node (ou null) Retourne le noeud du DOM contenant la page au chemin spécifié. Si le paramètre path n'est pas renseigné alors c'est la page active qui est retournée.
Evénements
Nom Details Description
router.biEvent { previous: PATH, current: PATH } L'événement est déclenché à chaque changement de page via la méthode goTo.
